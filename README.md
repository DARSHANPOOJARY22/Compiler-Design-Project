# Compiler Design Project

## Overview

This project is a simple compiler implementation developed in Python to demonstrate the fundamental concepts of compiler design. It showcases the basic phases involved in translating source code into a form that can be processed by a computer.

## Features

* Lexical Analysis
* Token Identification
* Syntax Processing
* Expression Evaluation
* Error Detection and Reporting
* Simple and Beginner-Friendly Implementation

## Technologies Used

* Python 3

## Project Structure

```
Compiler-Design-Project/
│
├── sum_compiler.py
└── README.md
```

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/DARSHANPOOJARY22/Compiler-Design-Project.git
```

2. Navigate to the project directory:

```bash
cd Compiler-Design-Project
```

3. Run the program:

```bash
python3 sum_compiler.py
```
